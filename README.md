# Description

This is the code of a game which applies the concept of bomber man game on the characters of pacman game. The idea is similar to that of bomber man. The player is required to demolish all the bricks in the playing arena while avoiding the ghosts.
Instead of a bomber man figure as the player's character, the game uses the pacman character.
